# 257.二叉树的所有路径

给定一个二叉树，返回所有从根节点到叶子节点的路径。

**说明:** 叶子节点是指没有子节点的节点。

**示例:**

```
输入：
    1
 2     3
   5
   
输出：["1->2->5", "1->3"]

解释: 所有根节点到叶子节点的路径为: 1->2->5, 1->3
```

**题解：**

```Python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    
    def binaryTreePaths(self, root: TreeNode) -> List[str]:
        def helper(root,path):
            if root:
                path += str(root.val)
                if not root.left and not root.right:
                    res.append(path)
                else:
                    path += "->"
                    helper(root.left,path)
                    helper(root.right,path)
        res = []
        helper(root,"")
        return res
         
        if not root:
            return None
        if root.left:
            return root.val
            self.binaryTreePaths(root.left)
        if root.right:
            return root.val
            self.binaryTreePaths(root.right)
        else:
            return root.val
```

传参机制，判断停止条件
